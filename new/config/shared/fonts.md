# Font Translation Patterns for Image Generation

This guide helps translate font names into visual descriptions that AI models can understand and reproduce in generated images.

## How to Use This Guide

When user requests a specific font:
1. Find the font in this list
2. Use the "Prompt Description" in your generation instructions
3. Include the "Visual Characteristics" for additional context

## USAGE GUIDELINES

### When Font Not in This List
1. Identify the category (geometric sans, humanist sans, serif, display, etc.)
2. Describe key visual characteristics
3. Use comparison: "similar to [known font] but [difference]"

### Combining Descriptions
For best results, use multiple descriptors:
- Category + Visual + Mood
- Example: "bold geometric sans-serif with extended width and impactful modern appearance"

### Greek Text Considerations
- Verify Greek support level before using
- "Excellent" = professionally designed Greek glyphs
- "Good" = functional Greek support
- "Limited" = basic Greek, may have issues

### Font Weights
Common weights to specify:
- Thin / Light
- Regular / Normal  
- Medium
- Semibold
- Bold
- Heavy / Black
- **Extended** (wider than normal)
- **Condensed** (narrower than normal)

### Example Prompt Integration
```
"Create marketing poster with text 'WINTER SALE' using 
clean geometric sans-serif with even stroke width and 
modern minimalist appearance (Montserrat style), 
bold weight, uppercase"
```
